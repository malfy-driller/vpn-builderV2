## Что выбрать

### Mixed
Основная сбалансированная подписка для большинства пользователей.

Подходит как первый вариант, если нужен один основной профиль без лишней ручной переборки.

### Pool
Расширенная подписка.

Содержит больше вариантов и подходит в тех случаях, когда хочется больший выбор конфигов или основной профиль временно работает хуже.

### LTE
Компактная подписка для мобильной сети.

Удобна для быстрых проверок на телефоне и для случаев, когда нужен небольшой профиль, который быстро обновляется и проще перебирается вручную.

---

## Обновление

Подписки обновляются автоматически.

Если ничего не подключается, рекомендуется:

- обновить подписку в приложении;
- проверить задержку и сортировку;
- попробовать авиарежим на 5–10 секунд;
- затем снова выбрать подходящий конфиг.

Подписку лучше обновлять:
- по Wi-Fi;
- или при уже активном VPN-соединении.

Работа конфигов может меняться со временем в зависимости от сети, оператора и маршрутизации.

---

## Использование

Добавьте нужную ссылку подписки в совместимый клиент, например:

- v2rayN
- v2rayNG
- Hiddify
- Happ

Если не хочется разбираться в деталях, начинайте с:

- **Mixed**

Если нужен дополнительный запас вариантов:

- **Pool**

Если нужен компактный профиль для мобильной сети:

- **LTE**

---

## Назначение проекта

Этот репозиторий нужен для автоматической сборки, обновления и публикации удобных подписок без лишней ручной работы.

Внутренние группы сборки используются как техническая база и не являются основными публичными профилями для повседневного использования.
