# VPN Builder 2

Набор автоматически обновляемых подписок для повседневного использования.

Проект собирает несколько профилей с разной логикой отбора конфигов и публикует их через GitHub Pages.

## Доступные подписки

### Mixed
Основная подписка для большинства пользователей.

- Сбалансированный и более чистый набор
- Подходит как первый вариант
- Обновляется автоматически

Ссылка:
`https://malfy-driller.github.io/vpn-builderV2/outputs/mixed.txt`

---

### Pool
Расширенная подписка.

- Более крупный набор конфигов
- Подходит, если нужен дополнительный выбор
- Может содержать больше альтернативных вариантов

Ссылка:
`https://malfy-driller.github.io/vpn-builderV2/outputs/pool.txt`

---

### LTE
Компактная подписка для мобильной сети.

- Небольшой быстрый профиль
- Удобно проверять и обновлять на LTE
- Подходит для быстрых тестов на телефоне

Ссылка:
`https://malfy-driller.github.io/vpn-builderV2/outputs/lte.txt`

---

## Что выбрать

Если нужен один основной вариант:

- **Mixed**

Если нужен более широкий выбор:

- **Pool**

Если нужен быстрый компактный профиль для телефона:

- **LTE**

---

## Обновление

Подписки обновляются автоматически.

Рекомендуется:
- обновлять подписку по Wi-Fi или при уже активном VPN-соединении;
- после обновления проверять задержку внутри клиента;
- выбирать рабочий конфиг через сортировку.

Если соединение не поднимается:
- включите авиарежим на 5–10 секунд;
- затем выключите его и попробуйте снова.

Работа конфигов может меняться со временем в зависимости от сети, оператора и маршрутизации.

---

## Использование

Добавьте нужную ссылку подписки в совместимый клиент, например:

- v2rayN
- v2rayNG
- Hiddify
- Happ

Для большинства случаев достаточно начать с:

- **Mixed**
