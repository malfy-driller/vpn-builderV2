# VPN Builder 2

Простой набор обновляемых подписок для повседневного использования.

Проект собирает несколько вариантов подписок с разной логикой отбора конфигов и обновляет их автоматически.

### Mixed
Основная универсальная подписка.

Подходит для большинства пользователей, если не хочется разбираться в деталях.  
Собирается из внутреннего пула серверов и периодически обновляется.

Рекомендуется как основной вариант для повседневного использования.

---

### White CIDR
Подписка на основе отдельной категории отобранных конфигов.

Может быть полезна как альтернативный вариант, если хочется сравнить работу разных подписок.

---

### White VLESS
Подписка на основе отобранных VLESS-конфигов.

Подходит как отдельный вариант для проверки и сравнения.

---

### Black All
Подборка из другой категории конфигов.

Иногда даёт полезные альтернативные варианты, если в других подписках что-то временно работает хуже.

---

### Pool
Внутренний пул конфигов.

Используется как промежуточная база для формирования основной подписки.  
Обычно не требуется для повседневного использования.

---

## Что выбрать

Если нужен один вариант без лишних вопросов, лучше использовать:

- **Mixed**

Если хочется вручную сравнить разные типы подписок, можно попробовать:

- **White CIDR**
- **White VLESS**
- **Black All**

---

## Обновление

Подписки обновляются автоматически.  
Количество конфигов и состав со временем могут меняться.

Если клиент поддерживает отображение `profile-title`, в названии подписки может показываться текущее количество конфигов.

---

## Использование

Добавьте нужную ссылку подписки в совместимый клиент и обновляйте её по мере необходимости.

Для большинства случаев достаточно одной основной подписки:

- **Mixed**
